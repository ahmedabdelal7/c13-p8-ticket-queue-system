## 📌 Description
A simple C++ system that simulates a real queue line used in service centers.  
The system issues tickets, manages waiting clients, and serves them using a FIFO queue.

## ⚙️ Features
- Issue new tickets
- Serve the next client
- Show queue information
- Display tickets in the queue
- Print all ticket details

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- `std::queue`
- `std::stack`
- Structs
- Console formatting

## 🏗️ Main Class
`clsQueueLine`

Responsible for managing tickets, queue operations, and client serving.
